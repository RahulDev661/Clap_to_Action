# Clap_to_Action
A simple and interactive Python desktop application that listens for a clap sound and automatically opens YouTube in Brave Browser. Built using audio processing and a Tkinter-based UI.

🚀 Features
👏 Detects clap using microphone
🌐 Opens YouTube instantly in Brave browser
🎚️ Adjustable sensitivity (to control detection accuracy)
🎬 Simple and clean UI using Tkinter
⚡ Real-time status updates (Listening / Detected / Stopped)
🧠 How It Works

The application continuously listens to microphone input and calculates the sound intensity.
When a sudden spike (clap) exceeds a defined threshold, it triggers an action to open YouTube.
